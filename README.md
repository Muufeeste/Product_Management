######Product Management System (PyWorks)#####

A Python and PyQt5 desktop inventory and member management system.

###Features
Add / Update / Delete products
Member management
Sell products to members
Automatic stock updates
Sales confirmation window
Statistics dashboard

##Technologies
Python
PyQt5
SQLite3

How to Run

Install requirements:

pip install PyQt5 pillow

Run:

python main.py

Notes
When it is first run, the database will be automatically created.
You have to change image path according to the place you download it so that you dont get error.
